# One Brain Two Hands
___
#### _Another small and probably useless project that makes me learn new stuff_
##### _[Still in progress...]_

### What it is
___
**One Brand Two Hands** is a mobile web application that helps users discover French associations with a social mission.
Users can explore associations located near them, in major French cities, and can use customizable filters 
(department, distance, associations category, ...).
They have also the possibility to save choices and preferences for future visits.

### Technological challenges
___
- **GSAP** animations
- Navigator **geolocation**
- Custom Symfony **UX Icons**
- **Leaflet**
- Leaflet.**SidePanel**
- Symfony **UX Map**
-  Symfony **Serializer**
- **Large database** migrations
- Custom Symfony **commands**
- **LocalStorage** and **sessionStorage** to save information without new Entities

### Familiar Technologies

- SCSS

- Font Awesome
